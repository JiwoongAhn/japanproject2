## 자세한 설명 / Full description (≤4000자)
```
UniOne（ユニワン）は、日本の大学生のためのキャンパスライフ・アプリです。
時間割・課題の管理、同じ大学の学生だけが使える掲示板、授業レビューを、ひとつのアプリで。

■ 時間割
・科目・曜日・時限を登録して、見やすい時間割表で管理
・今日の授業がひと目でわかる

■ 課題管理
・課題と締め切りを登録、提出状況を管理

■ 掲示板（学校別）
・同じ大学の学生だけが参加できる匿名掲示板
・質問・フリー・情報などカテゴリ別
・通報・ブロック機能で安心して利用できます

■ 授業レビュー
・授業の評価・コメントを共有

■ 在学生認証
・学校メールアドレスで在学生だけが登録できます

※ 本アプリは大学公式アプリではありません。学校システムの機能は、学生ご本人のログイン情報でのみ動作します。
```
