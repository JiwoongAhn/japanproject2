# Google Play 비공개 테스트(Closed Testing) 가이드 — A5

> 신규 **개인** 개발자 계정은 프로덕션 출시 전 **테스터 12명 / 14일 연속** 비공개 테스트가 필수입니다.
> (2024.12 정책 완화로 20명→12명. 2026 현재 유효. 법인 계정은 면제)
> 이 14일이 출시까지 가장 긴 대기 구간이므로 **테스터 모집을 가장 먼저** 시작하세요.

## 1. 핵심 규칙 (반드시 충족해야 프로덕션 신청 버튼 활성화)

| 항목 | 요건 |
|---|---|
| 테스터 수 | **최소 12명**이 비공개 테스트에 **옵트인(참여)** 한 상태 유지 |
| 기간 | **연속 14일** (중간에 테스터가 12명 미만으로 떨어지면 카운트 리셋 위험) |
| 활동 | 테스터가 실제로 앱을 설치/실행하는 것이 이상적 (단순 옵트인만으로도 카운트되나, 활동 있으면 안전) |
| 대상 | Google 계정(Gmail) 이메일. 본인 외 **순계정 12명** 필요(본인 계정은 카운트 모호 → 여유 있게 13~14명 권장) |

> ⚠️ **14일은 "옵트인한 테스터 12명이 유지된 날"부터 카운트**됩니다. 12명을 다 모은 그 시점이 D-day 시작이라, **모집을 빨리 끝낼수록 출시가 빨라집니다.**

## 2. Play Console 설정 단계 (빌드 완료 후)

1. **Play Console → 테스트 및 출시 → 비공개 테스트** 진입
2. **새 트랙 만들기** (또는 기본 "비공개 테스트" 트랙 사용)
3. **테스터 그룹 만들기**: "테스터" 탭 → 이메일 목록 그룹 생성 → 모집한 12~14명의 Gmail 주소 입력
4. **새 버전 만들기** → A4-2에서 빌드한 **AAB 업로드** (`eas build` 결과물)
5. **출시 노트(릴리스 노트)** 입력 (일본어 — 아래 템플릿 참고)
6. **검토 → 비공개 테스트 트랙으로 출시**
7. **옵트인 URL 복사** → 테스터들에게 배포 (이 링크로 들어가 "테스터 되기" 눌러야 설치 가능)

## 7. 테스터가 부족할 때 — 안전하게 셀프 확보하는 방법

> 주변 안드로이드 지인이 12명에 못 미칠 때를 위한 보충 가이드. (2026.6 리서치 기준)

### ⚠️ 절대 하지 말 것 (계정 정지 위험)

| 방법 | 왜 안 되나 |
|---|---|
| **본인 Gmail 여러 개로 채우기** | 같은 IP/기기 지문(fingerprint)을 Google이 **"인위적 부풀리기(artificial inflation)"** 로 감지 → 개발자 계정 플래그/정지 위험 |
| **에뮬레이터로 테스트** | **카운트 자체가 안 됨.** Google이 즉시 감지하고 실기기(physical device)만 인정 |
| **신규 Gmail 대량 생성 / 클라우드 기기 farm** | 위와 동일하게 감지 대상 |

→ "혼자 12개 계정"은 시간 아끼려다 **계정을 통째로 잃을 수 있는** 가장 위험한 길. 피한다.

### ✅ 안전하고 현실적인 방법 = 테스터 교환(상호 테스트) 커뮤니티

다른 개발자의 앱을 내가 테스트해주고, 그들이 내 앱을 테스트해주는 **P2P 상호 테스트**. 전부 실제 사람·실기기라 안전하고, 무료다.

| 채널 | 형태 | 비고 |
|---|---|---|
| **Testers Community 앱** | Google Play에서 설치 → closed-test 링크 등록 | 40,000+ 개발자, 무료, 가장 활발 |
| **Reddit** | r/TestersCommunity, r/AndroidAppTesters | 글 올리고 상호 테스트 |
| **Telegram** | @GooglePlay20Testers 등 그룹 | 빠른 회전 |
| **Facebook 그룹** | "Android Closed Testing" | 활동 많음 |
| **Discord / 포럼** | Android dev 서버 beta 채널, XDA, IndieHackers | 상호 피드백 |

- **핵심 = 상호성**: 내가 먼저 남의 앱을 테스트해주면 회수율이 올라간다.
- **소요 기간**: 무료 방식은 보통 **1~4주** (적극적으로 활동하면 단축).

### 💰 급하면 유료 서비스 (선택)

- **PrimeTestLab** ($14.99~, 4~6시간 내 시작), **20apptester.com**, **RealAppTesters**
- 사전 검증된 12~25명 제공. 단 비용·품질 편차가 있으니 후기 확인 후 사용.

### 🎯 추천 하이브리드 전략

1. **확보 가능한 안드로이드 지인 전원 먼저 락인** (한국 가족·친구 포함, 예: 4~6명) — 이탈 안 할 사람이라 14일 카운트의 안전판.
2. **부족분을 무료 교환 커뮤니티로 충당** (메인) — Testers Community 앱부터 시작.
3. **정말 급하면 유료로 보충**.
4. **테스터 국적·언어는 무관** — 12명/14일 요건 충족엔 글로벌 교환 OK. 일본인 실사용 피드백은 보너스, **요건 충족이 우선**.
